# Interactive Spotify TFT Display (ESP32)
A Spotify Display Module made with an ESP32 chip, with features like button control and interactive UI.
I added in some more features like mode changing using the buttons, between modes like the regular spotify display and music tracking, real time clock and system status monitoring. Those are the major improvements although im still thinking of adding a couple other things :)

I made this project mainly for learning purposes, cause ive never worked with hardware before, but there are other reasons as well, like im very interested in building hardware projects, even though i barely know anything (sounds funny right? haha). But seriously i like hardware stuff, so fully funded project plus im learning plus im having fun plusss prizes as well seems like a great reason to me!
